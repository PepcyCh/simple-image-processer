## MVVM

### Common

图像处理函数等

### Model

一个图像的 vector

### View

UI

### ViewModel

持有 Model（或 Model 的引用）

向 View 提供要绑定的函数，比如加载的函数、修改的函数

### Window / App

创建 M、V、VM 的对象，把 VM 提供的函数绑定给 V

把 M 给 VM，如果 VM 有的是引用的话
